HTML / CSS Cheat Sheet

HTML

HyperText Mark-up Language

code function
html starts an html document
body starts the body of an html document
h1-h6 headers, 1 is largest, 6 is smallest font
p this is used for a paragraph
section defines a section of a document
href links to a new page
target='_blank' use after href so that it opens in a new tab

CSS

Cascading Style Sheet

code function
font-family indicate font; be sure to use back up web-safe fonts when selecting specialty font

reminders in css

  • Use web-safe fonts as back up to specialty fonts.
